# Project Structure Documentation

## Overview
This document outlines the structure of the shopping intelligence tool project.

## Directory Structure

```
shopping-intelligence-tool/
├── docs/
│   └── PROJECT_STRUCTURE.md	 # Documentation for project structure
├── src/
│   ├── main/
│   │   ├── java/
│   │   └── resources/
│   ├── test/
│   │   ├── java/
│   │   └── resources/
├── lib/                  # External libraries
├── build/                # Build artifacts
├── config/               # Configuration files
└── README.md             # Project overview
```

## Details

- `docs/`
  - Contains all documentation related to the project.

- `src/`
  - `main/`
    - `java/`
      - Contains the main Java source code.
    - `resources/`
      - Contains resource files, such as configuration files or other assets.
  - `test/`
    - `java/`
      - Contains test source code.
    - `resources/`
      - Contains resource files used in testing.

- `lib/`
  - Contains external libraries used in the project.

- `build/`
  - Contains build artifacts generated from the build process.

- `config/`
  - Contains various configuration files needed for application settings.

- `README.md`
  - Provides an overview of the project including how to set it up and usage instructions.
